# Resumen del Proyecto: EMIL CLUB

## Descripción General
EMIL CLUB es una plataforma web interactiva y gamificada diseñada para los fans del artista Emil. El sitio ofrece una experiencia inmersiva con estética Cyberpunk/Neon, permitiendo a los usuarios registrarse, participar en sorteos y conectar con la comunidad.

## Estado Actual del Proyecto
El proyecto se encuentra en una fase avanzada de desarrollo con las siguientes características implementadas:

### Frontend (Visual & UI)
*   **Estética**: Diseño "Dark Mode" con acentos neón (verde `#ccff00`, morado, azul), efectos de vidrio (glassmorphism), ruido (noise) y animaciones fluidas.
*   **Hero Section**: Fondo animado con imagen (`FONDO1.png`) de alta visibilidad (opacidad 0.6), tipografía masiva y botón de llamada a la acción (CTA) pulsante.
*   **Bento Grid**: Cuadrícula interactiva con secuencias de imágenes animadas ("FANS" y "CONCIERTO") que cuentan con transiciones suaves de desvanecimiento (fade).
*   **Leaderboard (Sorteo del Mes)**: Panel que muestra el estado de los participantes, actualmente configurado para mostrar puntos XP.
*   **Formulario de Registro**: Formulario completo con validaciones, pasos múltiples (incluyendo consentimiento de menores) y un efecto de resplandor verde de fondo.
*   **Reproductor de Música**: Integración de Spotify con estilo personalizado acorde al sitio.

### Funcionalidad
*   **Sistema de XP**: Lógica para puntos de experiencia e interacción.
*   **Registro de Usuarios**: Integración con Supabase para el almacenamiento y gestión de datos de los fans.
*   **Animaciones**: Implementación robusta utilizando `framer-motion` para transiciones de página, efectos de scroll y micro-interacciones.

## Tecnologías Utilizadas
*   **Framework**: Next.js 14 (App Router)
*   **Lenguaje**: JavaScript / React
*   **Estilos**: Tailwind CSS
*   **Animaciones**: Framer Motion
*   **Iconos**: Lucide React
*   **Base de Datos**: Supabase
*   **Despliegue**: Vercel
*   **Control de Versiones**: GitHub
